## 介绍

从你的印象深刻 到他的集结成册

吴青峰《册叶一：一与一》

8首谱曲＋8首作词之选 当代华语乐坛词曲创作歌颂者

挑战单月个唱18场：《16叶》Legacy演唱会、云门舞集《上下册》

吴青峰《册叶一：一与一》 11/13 全册揭露

**序幕**

当一首首诗遇到一首首曲

就此翻开《册叶一：一与一》

每次的起心动念，都是一场自我内心剧场。吴青峰，拾起过往的创作，他是一位旅行的歌颂者，也是一位漂流的诗人。将那些你曾印象深刻的作品，集结成《册叶一：一与一》，上册是他谱出的音乐海洋，下册则是他写出的文字陆地。

**第一幕：海洋与陆地**

《册叶一：一与一》收录了吴青峰8首谱曲之作，8首作词之选，双CD以上下册的概念呈现，就像是一本历经时间的创作年谱，属于他的册叶：上册，是关乎空间的海上见闻故事集，他曾谱过的曲，配上诗人们的文字；从他第一首为人谱曲的作品〈费洛蒙小姐〉揭开序幕，接着，一幕幕海上剧场开演：〈我会我会〉、〈沙滩上的佛洛一德〉、〈最难的是相遇〉、〈阿兹海默〉、〈穿墙人〉、〈柔软〉，故事最终在挽歌般的〈低低星垂〉下交会。下册，是一场双脚踏出，关乎时间的旅程：不期而遇的〈宁静海〉、〈等〉、经历故事的〈年轮说〉、接着〈困在〉、进入土壤里的〈一点点〉、〈迷幻〉内的大块情感、〈月亮河〉，迎来最终章的〈极光〉；歌词内的绚烂与平淡，都是人生风景，生命的答案，其实那么简单。

**第二幕：那些关乎音乐的**

制作人：徐千秀、刘胡轶

在一开始的音乐制作，吴青峰以此张专辑：曲、词上下册的概念发展，各找了两位制作人负责。《太空人》专辑制作人徐千秀负责的是上册，制作人刘胡轶则是负责下册。在上册的制作面，由青峰的曲发展，找到许多优秀的编曲人：徐千秀、锺承洋，还有初次合作便将〈费洛蒙小姐〉的魅力展现的李端娴。跟这些编曲人合作时有个共同的特色，一开始，青峰跟制作人都没提供任何的参考资料，让编曲人们自由发挥，巧的是，他们竟都能还原青峰创作时脑中的那片海洋，如：〈最难的是相遇〉、〈柔软〉，甚至是透过这些编曲人，青峰找到了属于那些曲该有的样貌。

下册的制作，青峰当时写出这些词是因为听到创作人的曲，才衍生出词的故事，每个文字都像是音乐般，环抱着音符，也因此，青峰希望刘胡轶能先理解词，反向的从词去理解编曲应该是什么样貌；于是我们听到了〈一点点〉内的潮湿土壤与大鸣大放、〈迷幻〉里配器带出的纯粹情感、〈宁静海〉里月球的声响，还有许多的细腻，都藏在下册的音乐里，待你一一倾听。

**第三幕：诉说画面的一与一**

《一与一》概念贯彻到底

上下册平行时空诠释到位

《册叶一：一与一》的视觉概念，围绕在青峰的词曲创作所衍生出的画面，从专辑开案，就清楚地知道上下册各自需要的样貌，于是在每个环节上，都找到对应上下册完美配搭的一与一：MV导演黄中平v.s. 陈珊妮、歌词版MV导演涂皓钦v.s. 吴仲伦、摄影师登曼波与林建文、造型则由major tom与施筱柔各自负责。

在上册的谱曲部分，歌词以诗人的文字出发，画面充满了怪诞，由指标性导演黄中平开启这张专辑的MV视觉序幕，找到剧场导演黎焕雄提供最初期的脚本概念，中平导演出奇不意的大胆尝试，将剧场元素与音乐概念完美结合：〈费洛蒙小姐〉、〈最难的是相遇〉。下册的作词部分，由于青峰创作的歌词与生活各层面环环相扣，找到了青峰的好友，同时也是音乐制作人、导演陈珊妮来负责阐述那些词里的故事：〈一点点〉、〈迷幻〉。歌词版MV也毫不马虎的找到了《太空人》专辑负责〈译梦机〉及〈回音收集员〉的MV导演涂皓钦来负责上册；善于动画与故事阐述的导演吴仲伦负责下册。

摄影由艺术家登曼波与林建文负责，透过一个个画面呈现出专辑概念，有整理自己创作的图书馆员青峰、也有踏在满是植物森林里的单纯少年、更有期待海洋，等待漂流的那位歌者。专辑设计由谢富琇负责，实体专辑装帧左右开合的设计，代表了吴青峰脑内世界，那些关乎于曲的海洋、词的陆地，合而为一，融合得如此精彩。每个细节，都希望能完整《册叶一：一与一》的专辑概念，将青峰创作里的纯粹，呈现众人眼前。

**幕落，灯灭，音乐起**

舞台的幕落下，灯光熄灭，当万物都进入宁寂时，更多的音乐、文字，从宇宙的某个角落，再次浪漫的漂进他脑里，那些创作，将蛰伏于潮湿的土壤、晒进温暖的海洋，在适当的时机，再次成型，变成更多美好的创作，感动你我。他，华语乐坛难得一见的创作才子，吴青峰的《册叶一：一与一》，才刚翻起。

### 01 费洛蒙小姐 Ms Pheromone

> 這是他的序曲，曾有抗拒，隨之而來的密合又那麼沒有懷疑；在奇形怪狀的詞句之間以旋律神出鬼沒，旋律的抓痕憑空、荒謬、合理。而這竟然是他第一次替人譜曲的作品。 <br/>
> <br/>
> 16年前，〈費洛蒙小姐〉誕生於劇場導演徐堰鈴的劇作。16年後，青峰重唱〈費洛蒙小姐〉，徐千秀的新製作與李端嫻的新編曲，讓〈費洛蒙小姐〉的內心戲蔓延更加欲罷不能，歌曲也因著徐堰鈴導演原著的創意，採樣了周藍萍作曲，電影《梁山伯與祝英台》的〈十八相送〉。 <br/>
> <br/>
> 因著這首歌和劇場的淵源，〈費洛蒙小姐〉MV破天荒邀來三位導演的合作：奇幻詩意的腳本創意，來自劇場導演黎煥雄的發想；MV的拍攝與完成，由和青峰默契絕佳的黃中平導演挑起大樑；女主角費洛蒙小姐則由既是劇場導演，也是電影演員的陳雪甄出演；MV裡極具張力的群舞，由新生代的舞者與劇場演員共演。劇場意象加上視覺震撼，將曾經的〈費洛蒙小姐〉帶進數位，她16年後仍然蠻橫脆弱、歇斯底里，但沒有人能夠拒絕她旋律裡那不可理喻的魅力。 <br/>

> 我想把眼裡那些風景、身邊那些物件，和全部的自己 <br/>

### 03 沙滩上的佛洛一德 Freud on the Beach

### 05 一点点 Bit by bit

> 青峰說，「這是一隻蟬的故事。」2016年林宥嘉演唱的〈一點點〉，由林宥嘉作曲、吳青峰作詞、陳珊妮製作；其實當時青峰給出的第一版歌詞，就是現在2020年青峰版的〈一點點〉。這次的青峰版，音樂部分邀請了劉胡軼製作，編曲巧妙地表達出土壤內的生活，帶著一點點盼望與驚恐，以及那對出土後未知的迷惑；透過音樂刻劃出潮濕的土壤，點滴的水從外流入，跟著青峰的聲波，一起破土而出；就像所有的迷惘，都帶著無畏，隨著電吉他、鼓聲的躁動，大鳴大放。外面的世界，讓人驚惶，卻再也回不去土裡了。 <br/>

### 08 极光 Aurora

> 作為專輯的收尾，〈極光〉這首歌總和了整張專輯想說的話；聽過了上冊的海上故事集，踏過了下冊的文字風景，到了最終站的〈極光〉。寫這首歌時，青峰還沒看過極光，只是聽著好朋友Tanya蔡健雅滔滔不絕的講了她的極光之旅；接著，Tanya發了一首詞，希望青峰能把她看到的那些風景寫出。多年後青峰也終於看到極光，回頭看了自己寫過的歌詞，沒想到還真的就像文字裡那樣。視覺上，導演表現了等待極光的過程，日復一日的風景，只為那難得一刻的奇幻；極光之後的對白，卻又像是突然沒有了答案。 <br/>
